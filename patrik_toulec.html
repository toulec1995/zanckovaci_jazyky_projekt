XSD Type

  • string - v xml tagu s tímto typem tagu bude očekáván textový řetězec
  • date - v xml tagu s tímto typem bude očekáváno datum
  • integer - v xml tagu s tímto typem bude očekáváno celé číslo
  • byte - v xml tagu s tímto typem bude očekáváno jeden znak
